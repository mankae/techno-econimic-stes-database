# techno-economic-stes-database
repository containing CAPEX and OPEX cost data and access functions for PTES, TTES, BTES, and ATES, plus a simulation framework for simple heat-loss modeling of PTES and TTES—supporting cost analysis and basic performance simulation for seasonal thermal energy storage.

- [Cost function derivation](notebooks/CAPEX_OPEX_database_STES.ipynb)
- [Heat loss simulations of PTES](notebooks/Heat_Loss_Simulation_of_PTES.ipynb)
